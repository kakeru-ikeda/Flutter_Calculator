# Flutter_Calculator
Flutterで作成された電卓アプリケーションです。

Dartの練習がてら作成してるのでご愛嬌。

## Preview

<img src="https://github.com/kakeru-ikeda/Flutter_Calculator/assets/93127331/90843bdc-3f61-4f87-85d4-2d92cf8437e4" width="400px">

## Functions

- 括弧を利用した計算に対応します。
- パーセントを利用した計算に対応します。
- 小数点に対応します。
- ライトテーマ・ダークテーマの切り替えが可能です。
